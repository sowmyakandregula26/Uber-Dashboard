# 🚖 Uber Ride Booking Analytics Dashboard | Power BI

## 📌 Project Overview

This project is an interactive **Power BI Dashboard** designed to analyze Uber ride booking performance across the NCR region. The dashboard provides valuable insights into revenue, customer ratings, ride distance, booking trends, cancellation patterns, vehicle performance, payment methods, and location-wise ride activity.

The goal of this project is to help stakeholders make data-driven decisions by monitoring operational performance, customer satisfaction, and ride demand across different locations.

---

## 🎯 Business Problem Statement

Ride-booking platforms generate a large volume of booking data every day. Without proper analytics, it becomes challenging to identify operational inefficiencies, monitor customer satisfaction, and understand demand patterns.

This dashboard helps answer critical business questions related to:

* Revenue performance
* Ride completion and cancellation rates
* Customer satisfaction
* Vehicle utilization
* Payment preferences
* Location-wise demand analysis
* Operational efficiency

## ❓ Key Business Questions

### Booking Analysis

1. What is the total revenue generated from ride bookings?
2. Which vehicle type contributes the highest booking value?
3. What is the average customer rating?
4. How many rides were cancelled by customers and drivers?
5. How many rides remained incomplete?
6. What are the daily booking trends throughout the year?
7. Which payment method is most preferred by customers?
8. What is the total ride distance covered?

### Location Analysis

9. Which pickup locations have the highest booking demand?
10. Which drop locations receive the most rides?
11. Which locations generate the highest customer ratings?
12. How does booking status vary across locations?
13. Which vehicle types are preferred in different locations?
14. What is the average CTAT across locations?
15. Which locations require operational improvements?
